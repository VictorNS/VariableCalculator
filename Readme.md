# Variable calculator

Simple .Net application for calculating expressions with variables.

## Overview

This application provides a spreadsheet-like interface for defining variables and calculating expressions. You can create named variables, write mathematical expressions using those variables, and see the results instantly.

**Features:**
- Define variables with custom names (first column)
- Write expressions using variables and mathematical operators (second column)
- View calculated results automatically (third column)
- Add comments for documentation (fourth column)
- Add/remove rows as needed (action buttons)

## Technical Information

**Framework:** .NET 8.0 with Windows Presentation Foundation (WPF)
**Platform:** Windows
**Expression Engine:** DynamicExpresso.Core for mathematical expression evaluation
**UI Framework:** WPF with XAML for declarative UI design

**Requirements:**
- .NET 8.0 Runtime
- Windows OS
- Visual Studio 2022 or later (for development)
